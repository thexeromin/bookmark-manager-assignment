# Bookmark Manager

An assignment from Cometforge.

## Getting Started

First, set up your environment variables. Create a `.env` file in the root directory and add the required keys:

```bash
# Example .env file
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
